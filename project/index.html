# 项目简述

  • 在实际项目开发中使用过众多优秀的IDEA插件，引发了我对于插件开发的好奇，我能否也自己开发一些小插件来玩一玩？
  • 文件上传、下载......不同工具切来切去，文件拖来拖去，能不能都在IDEA中搞定？
  • Java代码热替换，Arthas命令复制又去服务器粘贴，贴上去还很长很长.......能不能不离开IDEA，鼠标点两下就搞定热更新？
  • 基于上述需求，我将一些常用的功能都集成到了这一个插件中，因此，这可能是一个不太符合“单一职责”的插件项目，但却是我练手和偷懒的好帮手 😉
  • 这个插件目前提供了以下功能，但很可能还会继续扩展
    • 将工程目录下指定文件上传到服务器，并在上传完成后指定指定命令
    • 将文件下载到指定工程目录下
    • 连接远程服务器，借助Arthas完成一键代码热更新
    • ......
